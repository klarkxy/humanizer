# 状态

## 当前目标
记录本次会话要推进的最小目标；类型、对象、状态必须在会话开始时填写，结束时更新。

类型：选填“立项 / 确认形态 / 补 canon / 做大纲 / 做章纲 / 写正文 / 修订 / 同步”等。

对象：具体指向，如“第 3 卷大纲”“主角 A 人物卡”“第 47 章正文”。

状态：当前进展，如“未开始 / 追问中 / 已确认 / 写作中 / 待同步”。

下一步：紧接着要执行的动作，让下次会话能直接接上下文。

## 当前门禁
列出进入当前目标下一阶段必须满足的条件；未全部通过时说明卡点所在。

## 最近上下文
记录最近 3–5 轮会话的关键结论；新会话先读这里，避免重复提问。

## 作者授权跳过门禁记录
记录作者明确同意跳过门禁的情况；注明跳过原因和承担的叙事风险。

## 最近同步
记录上一次从正文反推 canon 的时间、章节和主要变更；每次同步后更新。
