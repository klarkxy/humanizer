# 大纲

## 暂定标题

为什么修仙界没有开源组织？

## 预计字数

约 10000 字。

## 一句话梗概

年轻徒弟下山后听说凡人世界的开源，回山质问师父为什么修仙界不公开功法；师父在一夜雨谈里拆掉他的天真，指出功法不是书，组织不是牌匾，真正难的是让天下山门承认同一套验错和担责的秩序。

## 结构

### 1. 知乎题面与山雨归来（约 1000 字）

功能：建立知乎回答感和故事入口。

内容：

- 以题面切入：为什么修仙界没有 GitHub 之类的开源组织？
- 转入山中雨夜。徒弟刚从山下回来，衣上有雨，带着一种发现新事物的兴奋。
- 他没有熟练使用技术黑话，只说山下有许多人把自己写出来的东西公开，谁都能看，谁都能改，改得好还能还给原主。
- 师父一开始不接，先打哈哈：听着像善堂，又像祠堂门口贴偏方。
- 徒弟追问：为什么修仙界没有？为什么功法一定要锁在藏经阁里？

关键效果：

- 徒弟的第一层天真成立：公开等于天下共学。
- 师父老登味出场，但不要一开始就把话说尽。

### 2. 功法不是书，是入口（约 1800 字）

功能：拆掉“公开功法等于公开知识”的误解。

内容：

- 师父问徒弟：你知道源是什么吗？
- 徒弟答：源头。
- 师父纠正：源是入口。功法一旦被人照着练，就不是躺在纸上的字，而是进了经脉、丹田和神魂。
- 师父用药方、偏方、毒方作比：文字都一样是文字，但入口以后就不是文字。
- 徒弟说可以让大家一起看、一起挑错。
- 师父反问：谁看？炼气看筑基？筑基看金丹？散修看名门祖师批注？你看得懂字，不代表看得懂杀机。
- 展开修仙的“运行环境”问题：灵根不同，经脉宽窄不同，旧伤不同，前置心法不同，灵气环境不同。
- 同一篇清心诀，对一个人是清心，对另一个人可能是引心魔。

关键效果：

- 确立第一主因：功法无法无差别分发。
- 语气保持刻薄，但不堆技术词。

### 3. 公开之后，谁证明它是真的（约 1800 字）

功能：拆掉“放在一起让大家拿”的想象。

内容：

- 徒弟退一步：那至少可以把来路写清楚，谁传的、谁改的、谁练过，都记下来。
- 师父冷笑：你终于说到稍微像人的地方了。
- 进入来源和验印问题：
  - 玉简可以伪造。
  - 口诀可以少一个停顿。
  - 注释可以藏一个别人看不懂的禁制。
  - 一门功法可能三百年前就被某位掌门改过，只是后人当祖本供着。
- 徒弟说可以验。
- 师父问：谁验？昆仑验过，龙虎山认不认？蜀山封禁，南海散修服不服？
- 点出修仙界的山门林立：各家都有祖师、脸面、道统、禁忌和旧账。

关键效果：

- 确立第二主因：没有统一可信分发与验错机制。
- 师父并不是说不能验，而是说验完以后谁承认。

### 4. 那就建一个组织（约 1600 字）

功能：让徒弟提出第二层天真，也就是主动创建组织。

内容：

- 徒弟提出：那就建一个专门收功法、验功法、记来源、标禁忌、收事故的组织。
- 师父第一次不骂，点头说：主意不错。
- 徒弟愣住，短暂以为师父被说服。
- 师父继续：难的不是把功法放到一起。难的是天下山门愿意承认同一把尺。
- 展开裁决问题：
  - 你说某门心法有害，原门派说你毁他道统。
  - 散修练死了，名门说他资质不合。
  - 你说某真人改过玉简，他说你污他清名。
  - 你要封禁一门口诀，别人说你断他香火。
- 组织不是牌匾，而是共同承认的权威、流程、责任和羞耻感。

关键效果：

- 确立第三主因：没有共同治理与担责秩序。
- 师父不否定组织本身，只否定“建个名字就行”。

### 5. 西边魔网的对照（约 1700 字）

功能：明确使用魔网和密斯特拉，但只作为“组织背后需要什么”的对照。

内容：

- 徒弟问：难道天下没有人做成过？
- 师父说：西边那些法师，倒是接近。
- 讲魔网：
  - 法师的咒术不是各自塞在袖子里，有一张看不见的网承载。
  - 魔网由密斯特拉看守；咒式有痕，危险法术有限制。
  - 学院、塔、抄写员、法师组织围绕这张网形成秩序。
- 师父强调：那不是因为他们更善良，而是因为他们有一张天下法师大多不得不承认的网，还有一位女神压在上面。
- 简讲卡萨斯之灾：
  - 有人试图夺取整张网的权柄。
  - 浮城坠落，魔法失序。
  - 有秩序尚且出大祸。
- 重点落回修仙界：
  - 他们有网，有女神，有共同记录，尚且如此。
  - 修仙界有什么？玉简、口传、山门背书、祖师脸面。
  - 谁来做密斯特拉？昆仑？龙虎？蜀山？还是一个散修联盟？

关键效果：

- 魔网不做先进样板，而做对照。
- 说明真正难的是公共秩序和共同承认，不是西方魔法设定本身。

### 6. 闭门也会死人（约 1400 字）

功能：避免替闭源洗地，给主题厚度。

内容：

- 徒弟抓住反问：所以各门各派藏着就是对的？
- 师父沉默。第一次没有马上阴阳怪气。
- 师父承认：不是。
- 展开闭门之恶：
  - 散修拿不到全本，只能练残篇。
  - 小门派拿旁门口诀补正经经脉。
  - 有些禁忌害过人，却被锁进藏经阁。
  - 有些功法明明错了，各家怕丢脸，不说。
  - 名门的沉默也会杀人，只是死得慢，死得散。
- 徒弟问：那怎么办？
- 师父不提供建设性方案。

关键效果：

- 明确闭源无罪论被否定。
- 情绪从刻薄转冷。

### 7. 冷结尾：山里没有的是认错的人（约 700 字）

功能：收束题目和情绪。

内容：

- 雨声变大，山门外雾起。
- 师父说：你问为什么没有那样的组织。
- 徒弟沉默。
- 师父给最终答案：
  - 第一件事不是把功法拿出来。
  - 是让第一个害死人的门派承认是自己的功法害死了人。
  - 是让天下山门承认，同一本错书害死人时，死的不是散修自己的命。
- 徒弟不再问了。
- 最后一击：
  - 山里不是没有源。
  - 山里没有的是认错的人。

- “你看得懂字，不代表你看得懂杀机。”

- “你想建那样的组织，第一件事不是公开功法。”
- “山里不是没有源。山里没有的是认错的人。”
